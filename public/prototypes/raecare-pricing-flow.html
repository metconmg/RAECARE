Dual-Sided Marketplace
Two tracks.
One trust system.
RAECARE operates two parallel onboarding tracks that converge at the match layer. The caregiver side is a trust pipeline. The client side is a need intake system. They only meet when both sides are ready.

Pricing Model
Priced for
both sides.
RAECARE monetizes the caregiver side through a tiered subscription model built around verification depth and profile visibility. The client side pays for access to matches and ongoing ecosystem tools.
Caregiver · Tier 1
Verified
$29/mo
Background checked, identity confirmed. Profile visible in the network — not yet ranked for matches.
  • Identity + background verification
  • Basic profile listing
  • Network visibility (passive)
  • Secure document storage
  • Priority match ranking
  • Specialty certification badge
  • Direct client introductions
Caregiver · Tier 2
Certified
$59/mo
Full verification plus certified specialties. Priority placement in match engine. Recommended for active caregivers.
  • Everything in Verified
  • Competency assessment + certification
  • Specialty badges on profile
  • Priority match ranking
  • Up to 5 active matches/month
  • Client introduction access
  • Unlimited matches
Caregiver · Tier 3
Elite Network
$99/mo
Full certification plus unlimited match access, premium placement, and agency-level tools for independent operators.
  • Everything in Certified
  • Unlimited match connections
  • Top placement in results
  • Verified review collection
  • Performance analytics dashboard
  • Priority support + dispute resolution
  • Agency seat (coming soon)
Client · Tier 1
Search
$0/intake
Free need intake and match profile creation. No browsing until your profile is built and reviewed by the system.
  • Need intake form
  • Match profile creation
  • System compatibility scoring
  • Up to 3 match previews
  • Full caregiver profile access
  • Direct introduction
  • Ecosystem tools
Client · Tier 2
Connect
$49/mo
Full match access, caregiver introductions, and platform communication tools. Most families start here.
  • Everything in Search
  • Full caregiver profile access
  • Direct introduction + messaging
  • Scheduling tools
  • Secure payment processing
  • Care documentation log
  • Multi-caregiver coordination
Client · Tier 3
Care OS
$89/mo
Full ecosystem access for families managing complex or multi-caregiver care situations.
  • Everything in Connect
  • Multi-caregiver coordination
  • Shared family access (up to 4)
  • Care plan documentation
  • Incident + wellness logging
  • Emergency contact integration
  • Dedicated care coordinator (premium)
Revenue Architecture
Four streams.
One flywheel.
RAECARE's business model compounds as the network grows. Each new verified caregiver makes the platform more valuable to clients. Each paying client funds better caregiver tooling. The trust system is both the product and the moat.
Stream 01
Caregiver Subscriptions
$29–99/mo
Tiered monthly fees for profile listing, verification status, and match access.
Stream 02
Client Access Fees
$49–89/mo
Client-side subscriptions for match access, scheduling, payments, and ecosystem tools.
Stream 03
Transaction Layer
4–8%
Platform fee on payments processed between clients and caregivers.
Stream 04
Certification & Training
$79–149/cert
Certification assessments, continuing education modules, and specialty credential programs.
